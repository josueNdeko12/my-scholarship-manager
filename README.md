## ğŸ§ª How to Run Locally (Using XAMPP)

1. **Start Apache and MySQL** in the XAMPP Control Panel.

2. **Place the project folder** in the `htdocs` directory:  
   `C:\xampp\htdocs\my-scholarship-manager`
